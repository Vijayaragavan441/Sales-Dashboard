# Sales Dataset (Excel)

## Overview

This repository contains a Microsoft Excel sales dataset with 3,000 records. The dataset is organized and structured for data entry, analysis, reporting, and practice using Microsoft Excel.

## Dataset Details

* **File Format:** Microsoft Excel (.xlsx)
* **Total Records:** 3,000
* **Purpose:** Sales data analysis and reporting

## Dataset Includes

* Order Date
* Category
* Subcategory
* Sales
* Profit
* Region
* Quantity
* Customer Information

## Objectives

* Store sales records in a structured format.
* Practice Microsoft Excel data management.
* Perform sorting, filtering, and basic calculations.
* Create tables, charts, and reports in Excel.

## Software Used

* Microsoft Excel
